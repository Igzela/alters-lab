## Warnings

- Do NOT paste raw weekly notes into this template or the repo.
- Do NOT paste API keys.
- Do NOT paste raw provider prompts/responses.
- Only redacted summaries go into the repo.
- `synthetic: false` confirms this was a real note, not a test.
