3. Implement active_yaml.py

Required functions:

- default_project_root() -> Path
  Returns repository root from current file location.

- active_yaml_paths(project_root: Path | None = None) -> ActiveYamlPaths
  Returns canonical sealed baseline paths:
  - alters/current/snapshot.yaml
  - alters/current/branches.yaml
  - alters/current/alters/alter_A.yaml
  - alters/current/alters/alter_B.yaml
  - alters/current/alters/alter_C.yaml
  - alters/current/alters/alter_D.yaml
  - alters/current/value_alignment/alignment_2026-05-19.yaml
  - alters/current/dialogue/dialogue_alter_D_2026-05-19.yaml
  - alters/current/reality_trace.yaml

- load_yaml_file(path: Path) -> dict
  Read-only YAML load using yaml.safe_load.
  Raise FileNotFoundError if missing.
  Raise ValueError if file does not parse to dict.

- load_active_yaml_chain(project_root: Path | None = None) -> ActiveYamlChain
  Loads all 9 active artifacts.
  Does not write anything.

- validate_active_yaml_chain(chain: ActiveYamlChain) -> ValidationResult
  Validate:
  snapshot:
    - top-level snapshot exists
    - intake_status.phase == completed
    - pending_anchor is null
    - all three anchors non-empty
  branches:
    - branch_discovery.status == completed
    - 4 branches exist
    - ids are branch_A, branch_B, branch_C, branch_D
    - incompatible_with non-empty for every branch
  alters:
    - 4 alters exist
    - each has id matching alter_A-D
    - branch_ref matching branch_A-D
    - generated_at == 2026-05-19
    - time_horizon == 1.5-2年后
    - source_refs.snapshot_ref == alters/current/snapshot.yaml
    - source_refs.branches_ref == alters/current/branches.yaml
    - source_refs.rubric_ref == alters/calibration/rubric.yaml
    - quality_status.human_confirmed is true
    - quality_status.active is true
  value_alignment:
    - value_alignment_report.status == human_confirmed
    - final_interpretation.primary_candidate == branch_D
    - provisional_commitment.selected_branch == branch_D
  dialogue:
    - dialogue.status == human_confirmed_static_artifact
    - session.alter_ref == alters/current/alters/alter_D.yaml
    - context_policy.provider_used is null
    - context_policy.runtime_used is false
  reality_trace:
    - reality_trace.status == active
    - current_probe.selected_branch == branch_D
    - current_probe.day_14_gate.status == completed
    - current_probe.day_30_gate.status is passed/completed/pending, but must exist
    - calibration_readiness.ready_for_reality_score is false unless explicitly changed by a later approved slice

  Return ValidationResult with errors/warnings, not exceptions, for semantic validation failures.

- summarize_active_yaml_chain(chain: ActiveYamlChain) -> dict
  Return a read-only summary:

    "branch_count": 4,

    "calibration_ready": false
  }
